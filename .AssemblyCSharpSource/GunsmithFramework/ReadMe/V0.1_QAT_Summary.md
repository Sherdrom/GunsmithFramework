# V0.1 QAT Summary

日期：2026-05-20

QAT = Quick Attachment Transform。本轮目标是执行 `QuickAttachmentTransformPlan.md` 的低风险前半段：先建立 GunSmith 自己的 quick-slot 附件 transform 查询层，然后让 DeepLaser 和 quick-slot 灯光优先读取该 transform，减少对真实 hidden `ItemContainer` 物理位置的依赖。

## 已完成内容

- 新增 `GunsmithQuickAttachmentTransformService`。
  - 可通过附件本身、武器+附件、或武器+附件+slot index 查询 quick-slot transform。
  - 输出 world/draw position、world/draw rotation、方向向量、朝向、submarine、current hull。
  - 查询失败时返回 `false`，不修改 item/body/rect。
- 保留现有 quick-slot layout 兼容行为。
  - `DeepGunsmithRegisterQuickSlotLayout` 和 `DeepGunsmithApplyQuickSlotLayouts` 未改名。
  - `GunsmithQuickSlotLayoutPatch` 仍会在 `Item.SetContainedItemPositions()` 后应用现有 physical layout。
  - 旧 physical layout 现在复用 transform service 的计算结果，避免位置算法分叉。
- 迁移 DeepLaser。
  - DeepLaser draw path 不再调用 `weaponItem.SetContainedItemPositions()`。
  - 如果附件能查询到 GunSmith transform，激光起点使用 transform draw position，方向使用 transform direction。
  - 如果 GunSmith API 不存在或查询失败，继续回退到原本的 `laserItem.DrawPosition` + weapon body direction。
  - DeepLaser 对 GunSmith 使用反射软依赖，避免新增硬项目引用。
  - 反射调用和 quick-slot body transform 不使用 `try/catch` 吞异常；异常会直接暴露，便于早期定位。
- 迁移 quick-slot `LightComponent`。
  - 新增 `GunsmithQuickSlotLightPatch`。
  - 只在 light item 能查询到 GunSmith quick-slot transform 时覆盖光源位置和旋转。
  - 在 `SetLightSourceTransform()` 和 `LightComponent.Update()` 后刷新，确保移动/瞄准时光源持续跟随附件。
  - 光源 parent 到武器 body，并用 transform draw position 计算 `OffsetFromBody`；这样隐藏 quick-slot 内的手电不会被原生可见性判断错误关闭。
  - 当附件离开 GunSmith quick slot 时，清除 patch 设置过的 `ParentBody` 并恢复原生 light transform。
  - 非 GunSmith quick-slot 灯光保持 Barotrauma 原生行为。

## 游戏内观察点

- HK416 QuickMod：
  - muzzle 单击安装/拆卸后附件不消失。
  - slot 1-5 都能拖出、拖回。
  - 不兼容附件拖入失败后原状态保持正常。
- DeepLaser：
  - 激光起点贴合可见 laser device。
  - 左右朝向都正确翻转。
  - 走动、瞄准、站在潜艇内外时起点持续跟随附件。
  - 非 GunSmith 武器继续使用原生行为。
- LightComponent：
  - 手电/激光灯光源从可见附件位置发出。
  - 左右朝向和移动瞄准时持续跟随。
  - 非 GunSmith quick-slot 灯光不受影响。
- 控制台不应出现：
  - `DeepLaser failed to resolve GunSmith quick attachment transform API`
  - `DeepLaser received an invalid GunSmith quick attachment transform payload.`
  - `DeepLaser received a non-finite GunSmith quick attachment transform.`
